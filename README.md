#Livro de Receitas

 ## Requisitos

 - Usuário e Senha
 - Cadastro de receitas
 - Cada receita terá vários ingredientes (ingredients)
 - Cada receita terá várias instruções (directions)
 - Cada receita terá uma imagem

## Dados

  - Receita: Título, descrição, Imagem
  - Ingredientes: Nome
  - Instruções: Descrição
